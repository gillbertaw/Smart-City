# 🗳️ Sistem Voting
> Sebuah sistem voting berbasis web yang dirancang untuk membantu dalam mengambil keputusan secara transparan, efisien, dan aman.

---

## 🎯 Deskripsi Project

Project ini dikembangkan menggunakan metode **Agile** untuk membangun platform voting yang sederhana namun efektif.
Sistem ini memastikan keadilan dalam voting dengan konsep **1 user = 1 suara**.
